A Beginner-Friendly Guide to How the MetaQR App Works

1. The Big Picture 🗺️

This project is built using the three core building blocks of the web: HTML, CSS, and JavaScript. Here is how they collaborate:

  • HTML (The Skeleton): Defines what elements are on the page (the input fields, the title, the QR image box, and the buttons).
  • CSS (The Styling): Makes it look beautiful. It sets the dark gradient background, colors, layout columns, and the glowing buttons.
  • JavaScript (The Brain): Adds behavior. It listens to what you type, sends it to a server to make a QR code, and triggers the download or print actions.

2. How QR Code Generation Works ⚙️

Instead of using complex local files to calculate and draw a QR matrix mathematically, this app uses a free public **API** (Application Programming Interface).

🍔 The Restaurant Analogy (What is an API?)

Think of an API like ordering food at a restaurant:

  1. You (the Browser) are the customer. You tell the waiter what you want (your text and size).
  2. The API URL is the order ticket written by the waiter. It says exactly: "We need a QR image containing 'metamask.io' at size 250px."
  3. The API Server is the kitchen. It processes your request, generates the QR code image, and sends it back to your table (displays it on the screen).

In `script.js`, the code constructs this order ticket URL dynamically:

qrImg.src = `https://api.qrserver.com/v1/create-qr-code/?data=${encodeURIComponent(text)}&size=${size}x${size}`;

What this does:

  • data=... holds the text you want inside the QR code.
  • size=... specifies the width and height (e.g., 250x250).
  • encodeURIComponent(text) is a built-in helper that changes special characters (like ?, &, or spaces) into a format safe for URLs (for example, a space becomes %20).

3. Real-Time Generation (Event Listeners) ⚡

How does the QR code update instantly while you type? We use Event Listeners.

inputText.addEventListener('input', updateQRCode);
qrSize.addEventListener('input', updateQRCode);

An Event Listener is like a guard watching the input boxes. Every time the user presses a key, deletes characters, or changes the size number, the guard shouts: "Hey, the input changed! Run the updateQRCode function right now!". This changes the image source instantly, and the browser displays the new QR code.

4. How Downloading Works 💾

The app offers two ways to save the QR code depending on the dropdown selection: PDF Document or PNG Image.

📄 A. Downloading as a PDF Document

When the user selects PDF and clicks the button, JavaScript triggers: window.print();

This command opens the standard browser print dialog. To make this act like a PDF download, we added special **print styles** to `index.css`:

@media print {
  body * { visibility: hidden; }
  .qr-preview-box, .qr-image-container, #qrImg { visibility: visible; }
  .qr-preview-box { position: absolute; left: 50%; top: 50%; transform: translate(-50%, -50%); }
}
  • @media print: This CSS rule runs only when you print the page or save it as a PDF.
  • body * { visibility: hidden; }: Hides everything on the page (inputs, buttons, titles, dark backgrounds).
  • visibility: visible: Forces only the QR code preview to remain visible.
  • position: absolute; left: 50%...: Centers the QR code perfectly on the sheet of paper.

As a result, in the print preview dialog, you see a clean, white sheet with only the QR code centered, ready to be printed or saved directly as a PDF!

🖼️ B. Downloading as a PNG Image

Since the QR code is generated by an external server (`api.qrserver.com`), security boundaries inside browsers block simple direct file downloads (this is called cross-origin restriction).

To make the download simple and cross-origin safe without using complex binary fetches, JavaScript triggers:

window.open(qrImg.src, '_blank');

This line opens the QR code image directly in a new browser tab. Once it's open, the user can easily right-click on the image and click "Save Image As..." to download it to their device. This is highly reliable, uses only one line of code, and is very simple to explain!
